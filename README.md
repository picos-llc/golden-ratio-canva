# 🎨 <ruby>絵画構図<rt>かいがこうず</rt></ruby>アシスタント・クロッパー

こんにちは！これは、きみの写真やイラストを**「かっこいい<ruby>構図<rt>こうず</rt></ruby>（ならべ方のルール）」**で切りとる（トリミングする）ための、魔法のような道具だよ！

---

## 🌟 これでなにができるの？

### 1. 🖼️ 写真や絵をきれいに切りとれる！
いらないところを切りとって、自分が一番見せたい形や大きさ（正方形やワイド画面など）にできるよ。

### 2. 🤖 AI（人工知能）が「ここがポイント！」と教えてくれるよ！
画像をのせると、AIが自動で次の2つのポイントを見つけてくれるよ。
* **🔍 <ruby>詳細<rt>しょうさい</rt></ruby>の焦点（エッジ）**：絵の中で、細かくかかれている場所（主役になりやすい場所）だよ。
* **☀️ <ruby>光明<rt>こうみょう</rt></ruby>の焦点（明るさ）**：絵の中で、一番ピカッと光っている明るい場所だよ。

このポイントの近くに主役をおくと、とってもバランスの良い絵になるよ！

### 3. 📐 プロの画家が使う「魔法の線」を重ねられる！
昔のすごい画家や写真家が使っていた、絵がかっこよく見える「線のパターン」を、きみの絵の上に重ねて見ることができるよ！

* **三分割法（さんぶんかつほう）**：絵をタテ・ヨコに3つに分けて、線の交わるところに主役を置く方法。
* **黄金螺旋（おうごんらせん）**：カタツムリのぐるぐるみたいな渦巻き（うずまき）の線に合わせて、人の目線をあつめる方法。
* **動的対称性（どうてきたいしょうせい）**：ななめの線を使って、絵に動きやかっこよさを出す方法。
* **黄金三角形（おうごんさんかくけい）**：さんかくの形を使って、すっきりとしたバランスにする方法。

## 🚀 つかいかた

このアプリは、インターネットにつながっていなくてもうごくよ！きみのパソコンの中だけで処理するから安心だよ。

1. **アプリをひらく**
   `index.html` というファイルをダブルクリックするか、ブラウザ（Chromeなど）に引っ張って（ドラッグ＆ドロップして）開いてね。
2. **絵をえらぶ**
   「ファイルを選択」ボタンをおして、切りとりたい写真や絵をえらんでね。
3. **かっこいい線をえらぶ**
   右がわにあるメニューから、好きな「魔法の線」や、切りとる形（正方形や、黄金比の長方形など）をえらんでね。
4. **切りとる場所を決める**
   絵の上の白い枠（わく）の角をマウスで動かして、きれいに見える場所を探してね。
5. **保存（ほぞん）する**
   「構図を決定して保存する」ボタンをおすと、切りとった新しい画像がパソコンに保存されるよ！
